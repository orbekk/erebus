Erebus

Erebus is a CalDAV server designed to support conversion from and to different backends. It is targeted towards Microsoft Exchange, so you can use your Microsoft Exchange calendar in open source calendar applications, like Mozilla Sunbird and Evolution.

Source code

The source code can be fetched from the main git repository:

$ git clone git://github.com/orbekk/erebus.git

You can also browse the source code on gitweb or download a tarball of the current revision
